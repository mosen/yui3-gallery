Class Y.DP.TableBase - extends Y.Widget

Dynamic table class.

Properties

_ioHandlers - Object

Object used for IO callback. Contains four functions to handle each stage of the IO request.

_tbodyNode - Node

Reference to the TBODY node containing this table data.

CELL_TEMPLATE - String

Standard cell template.

CellRenderers - object

Cell renderers are implemented as functions which return closures that are seeded with the configuration. Example closure: supplying a width parameter returns a function that renders a cell, and contains that width variable 'pre-seeded'.

cells - object

Array of cells to render. Does not necessarily have a 1:1 relationship with DataSource fields. cells are specified in the format { field: "fieldname", renderer: fnCellRenderer }

CONTENT_TEMPLATE - String

Body section does not require a contentBox because the content is bounded by the TBODY node.

ROW_TEMPLATE - String

Standard row template.

Widget.ATTRS - static Object

Static property used to define the default attribute configuration for the Widget.

Widget.NAME - static String

Static property provides a string to identify the class.

Currently used to apply class identifiers to the bounding box and to classify events fired by the widget.

ZEROROWS_TEMPLATE - String

Row template for a status message, that spans the entire table.
