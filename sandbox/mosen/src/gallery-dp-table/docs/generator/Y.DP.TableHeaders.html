Class Y.DP.TableHeaders

Table Column Headers Progressive enhancement of TH elements to provide sorting functionality.

Properties

COLUMN_LABEL_TEMPLATE - static String

Column text content will be wrapped in this element.

COLUMN_SORT_INDICATOR_TEMPLATE - static String

Contains the sort indicator graphic

CONTENT_TEMPLATE - static object

No content template because this is encompassed by a table tag.

Configuration Attributes

columns - Array

Array of objects representing columns with keys for title and width This will be refactored into a columns object.
Default Value: Empty array

columnsNode - Node

Reference to the table row element containing the column headers
Default Value: null

theadNode - Node

Reference to the table head element
Default Value: null
