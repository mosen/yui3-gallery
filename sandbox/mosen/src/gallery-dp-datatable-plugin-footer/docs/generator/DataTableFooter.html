Class DataTableFooter - extends Y.Base.Plugin

The DataTable footer plugin allows you to display any value in the footer section of the table. You can do simple calculations or provide a function to render content based on the recordset. The footer columnset definition is different to the DataTable columnset definition because there are use cases for footer columns not based on data, or aggregated from multiple columns.

Configuration Attributes

columnset - Array

A couple of example values for this configuration attribute:

One column that contains only the word "String" but spans 2 columns of the table.
[ { value: "String", span: 2 } ]

One column that executes a function to get its contents, the function takes the RecordSet instance of the Y.DataTable as its only parameter
[ { value: fnCalculateTotal } ]

tdTemplate - String

Template for every footer cell

tfootTemplate - String

Template for the footer minus the columns
